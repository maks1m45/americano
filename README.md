# Americano · генератор турниров по паделу

Одностраничное веб-приложение (один HTML-файл) для проведения турниров по паделу в формате **Americano**: генерация пар и раундов, ввод счёта, рейтинг по разнице очков, автосохранение и экспорт JSON. Полностью на клиенте, без сервера.

**Живая версия:** https://maks1m45.github.io/americano/

## Возможности
- Настройка: название, число кортов, очков до победы, список игроков (мин. 4)
- Автогенерация расписания: меняющиеся пары, равномерный отдых, распределение по кортам
- Рейтинг по разнице очков, обновление в реальном времени
- Пьедестал топ-3, прогресс турнира, аватары игроков, конфетти на финише
- Тёмная/светлая тема, адаптив от 375px
- Автосохранение в localStorage, экспорт/импорт JSON

Открывается двойным кликом по `index.html` в любом браузере — работает офлайн.
